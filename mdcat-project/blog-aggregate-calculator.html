Aggregate Calculator Guide — How It Works

4 min read

Every university in Pakistan uses a different formula to calculate your aggregate. Here's a complete breakdown.

UHS (Punjab) Formula

Aggregate = (Matric % × 10%) + (FSc % × 40%) + (MDCAT % × 50%)

Example: Matric 95%, FSc 92%, MDCAT 185/200 (92.5%)
= (95×0.10) + (92×0.40) + (92.5×0.50) = 9.5 + 36.8 + 46.25 = 92.55%

KMU (KPK) Formula

Aggregate = (Matric % × 10%) + (FSc % × 40%) + (MDCAT % × 50%)

NUMS Formula

Aggregate = (Matric % × 10%) + (FSc % × 40%) + (NUMS Test % × 50%)

Aga Khan University

AKU uses its own entrance test (AKUEB) — MDCAT score is not accepted.

Tips to Improve Your Aggregate

  • Every mark in MDCAT counts — 1 mark = 0.5% in aggregate
  • FSc marks have 40% weightage — don't neglect them
  • Target 185+ in MDCAT for top colleges
